# M03704:扩号匹配问题
**核心算法：栈**
```python
while True:
    try:
        string=input()
        output=[' ']*len(string)

        #一个个字符检查，有'('入栈，有')'出栈
        stack=[]
        for i in range(len(string)):
            if string[i]=='(':
                stack.append(i)
            elif string[i]==')':
                if stack:
                    stack.pop()
                else:
                    output[i]='?'
        for i in stack:
            output[i]='$'

        print(string)
        print(''.join(output))
    except EOFError:
        break
```
