# E18223:24点
**算法核心：条件语句**
```python
m=int(input())
for _ in range(m):
    a,b,c,d = map(int, input().split())
    
    signs=[1, -1]
    found=False
    # 四重循环：分别枚举a、b、c、d的正负
    for sa in signs:
        for sb in signs:
            for sc in signs:
                for sd in signs:
                    if sa*a+sb*b+sc*c+sd*d==24:
                        found=True
                        break 
                if found:
                    break
            if found:
                break
        if found:
            break
    print("YES" if found else "NO")
```
