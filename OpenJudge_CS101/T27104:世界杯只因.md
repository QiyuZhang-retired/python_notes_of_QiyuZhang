# T27104:世界杯只因
**算法核心：区间覆盖**
```python
N=int(input())
a=list(map(int,input().split()))
intervals=[]
for i in range(1,N+1):
    intervals.append((i-a[i-1],i+a[i-1]))

intervals.sort()
curr_end=0
next_end=0
cnt=0
i=0

while curr_end<N:
    #离散覆盖，要加一
    while i<N and intervals[i][0]<=curr_end+1:
        next_end=max(next_end,intervals[i][1])
        i+=1

    cnt+=1
    curr_end=next_end

print(cnt)
```
