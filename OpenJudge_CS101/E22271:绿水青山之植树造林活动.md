# E22271:绿水青山之植树造林活动
**核心算法：字典**
```python
N=int(input())
trees={}

for _ in range(N):
    tree=input()
    if tree in trees:
        trees[tree]+=1
    else:
        trees[tree]=1

for tree in sorted(trees):
    print(f'{tree} {100*trees[tree]/N:.4f}%')
```
