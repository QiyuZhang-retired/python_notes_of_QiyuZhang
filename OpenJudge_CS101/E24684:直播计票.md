# E24684:直播计票
**核心算法：字典**
```python
from collections import Counter

votes=list(map(int,input().split()))
cnt=Counter(votes)

max_cnt=max(cnt.values())
max_votes=sorted([votes for votes,num in cnt.items() if num==max_cnt])

print(' '.join(map(str,max_votes)))
```
