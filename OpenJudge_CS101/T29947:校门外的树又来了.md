# T29947:校门外的树又来了
**核心算法：区间合并**
```python
L,M=map(int,input().split())
removes=[]
for _ in range(M):
    a,b=map(int,input().split())
    removes.append((a,b))
removes.sort()

#进行区间合并
processed_removes=[]
l=removes[0][0]
r=removes[0][1]
for n in removes:
    if r>=n[0]:
        r=max(r,n[1])
    else:
        processed_removes.append((l,r))
        l=n[0]
        r=n[1]
processed_removes.append((l,r))

result=L+1
for m in processed_removes:
    result-=m[1]-m[0]+1
print(result)
```
