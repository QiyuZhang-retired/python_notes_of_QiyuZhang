# E18224:找魔数
**核心算法：条件语句**
```python
m=int(input())
nums=list(map(int,input().split()))

for num in nums:
    for a in range(1,int(num**0.5)+1):
        b=int((num-a**2)**0.5)
        if b**2+a**2==num and b>0:
            print(bin(num),oct(num),hex(num))
            break
```
