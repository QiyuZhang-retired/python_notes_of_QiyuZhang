# M18182:打怪兽
**核心算法：哈希表**
```python
N=int(input())
for _ in range(N):
    n,m,b=map(int,input().split())
    hash_dict={}
    for i in range(n):
        ti,xi=map(int,input().split())
        if ti not in hash_dict:
            hash_dict[ti]=[xi]
        else:
            hash_dict[ti].append(xi)

    sorted_key=sorted(hash_dict)
    output='alive'
    for key in sorted_key:
        if m>=len(hash_dict[key]):
            b-=sum(hash_dict[key])
        else:
            hash_dict[key]=sorted(hash_dict[key],reverse=True)
            b-=sum(hash_dict[key][:m])

        if b<=0:
            output=key
            break

    print(output)
```
