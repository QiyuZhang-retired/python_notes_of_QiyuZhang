# E20742:泰波拿契數
**算法核心：递推**  
n较小，直接递推
```python
def T(n):
    if n==0:
        return 0
    if n==1:
        return 1
    if n==2:
        return 1
    else:
        a,b,c=0,1,1
        for _ in range(n-2):
            a,b,c=b,c,a+b+c
        return c

n=int(input())
print(T(n))
```
