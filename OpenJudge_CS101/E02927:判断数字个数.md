# E02927:判断数字个数
**算法核心：字符串**
```python
while True:
    try:
        word=input()
        nums=[]
        for c in word:
            if c.isdigit():
                nums.append(int(c))
        for i in range(10):
            if nums.count(i)!=0:
                print(f'{i}:{nums.count(i)}')
    except EOFError:
        break
```
