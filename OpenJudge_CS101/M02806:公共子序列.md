# M02806:公共子序列
**核心算法：动态规划（LCS）**  
1）状态转移函数：用二维数组表示X前i项和Y前j项最长公共子序列的长度  
2）状态转移关系：若此处两项相等，那应该是去掉两项后的最大长度加一；若不相等，应该是去掉两项之一的最大长度的较大值
```python
#不定行输入
while True:
    try:
        X,Y=input().split()

        #设置状态转移函数
        x,y=len(X),len(Y)
        dp=[[0]*(y+1) for _ in range(x+1)]

        #进行状态转移
        for i in range(1,x+1):
            for j in range(1,y+1):
                if X[i-1]==Y[j-1]:
                    dp[i][j]=dp[i-1][j-1]+1
                else:
                    dp[i][j]=max(dp[i-1][j],dp[i][j-1])

        print(dp[x][y])
    except EOFError:
        break
```
