# M03532:最大上升子序列和
**核心算法：动态规划（LIS）**
```python
N=int(input())
nums=list(map(int,input().split()))

#设置状态转移函数并进行转移
dp=nums[:]
for i in range(1,N):
    for j in range(i):
        if nums[j]<nums[i]:
            dp[i]=max(dp[i],dp[j]+nums[i])

print(max(dp))
```
