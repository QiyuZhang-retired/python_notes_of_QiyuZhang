# E04138:质数的和与积
**算法核心：质数**  
数据较小，直接试除
```python
import math
def is_prime(n):
    if n<=1:
        return False
    if n==2:
        return True
    if n%2==0:
        return False
    for i in range(3,int(math.sqrt(n))+1,2):
        if n%i==0:
            return False
    return True

S=int(input())
nums=[]
for i in range(2,S//2+1):
    if is_prime(i) and is_prime(S-i):
        nums.append(i*(S-i))
print(max(nums))
```
