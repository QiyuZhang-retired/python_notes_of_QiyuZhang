# M04030:统计单词数
**算法核心：字符串**  
加入空格保证所有边界匹配情况都可以正常运行
```python
origin_word=input().lower()
origin_sentence=input().lower()

#加入空格使其完全匹配
word=' '+origin_word+' '
sentence=' '+origin_sentence+' '

#使用find方法找到第一处位置（没找到自动返回-1）
first=sentence.find(word)

if first==-1:
    print(-1)
else:
    count=0
    current=0

    while True:
        place=sentence.find(word,current)
        if place==-1:
            break

        count+=1
        current=place+1

    print(count,first)
```
