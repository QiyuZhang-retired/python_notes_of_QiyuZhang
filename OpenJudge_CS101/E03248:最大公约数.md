# E03248:最大公约数
**算法核心：for循环**  
试除法，注意不定行输入
```python
while True:
    try:
        a,b=map(int,input().split())
        nums=[]
  
        for i in range(1,a+1):
           if a%i==0 and b%i==0:
               nums.append(i)

        print(max(nums))
    except EOFError:
        break
```
