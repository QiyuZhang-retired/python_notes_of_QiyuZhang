# M02746:约瑟夫问题
**算法核心：队列**
```python
from collections import deque

while True:
    n,m=map(int,input().split())
    if n==m==0:
        break

    queue=deque()
    for i in range(1,n+1):
        queue.append(i)

    #队列模拟圈
    while len(queue)>1:
        for _ in range(m-1):
            queue.append(queue.popleft())
        queue.popleft()

    print(queue.popleft())
```
