# M18211:军备竞赛
**核心算法：贪心算法，双指针**
```python
p=int(input())
prices=list(map(int,input().split()))
prices.sort()

weapons=0
i,j=0,len(prices)-1

while i<=j:
    if p>=prices[i]:
        p-=prices[i]
        i+=1
        weapons+=1
    elif weapons>=1 and i!=j:
        p+=prices[j]
        j-=1
        weapons-=1
    else:
        break
        
print(weapons)
```
