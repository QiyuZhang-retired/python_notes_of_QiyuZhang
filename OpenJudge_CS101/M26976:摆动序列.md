# M26976:摆动序列
**算法核心：贪心算法**  
1）用一个变量记录当下摆动方向  
2）在每一处有两种可能：与摆动方向相反，进行状态改变，继续遍历；其他情况（摆动方向相同或不摆动），直接更新位置，保证更小（更大），避免与后续比较出错
```python
n=int(input())
nums=list(map(int,input().split()))
if n<=1:
    print(n)
    exit()

#判断是否有长度不为一的序列，并排除前面重复的数字，方便设置初始状态
current=0
while current<n-1 and nums[current]==nums[current+1]:
    current+=1
if current==n-1:
    print(1)
    exit()

#设置初始状态
is_bigger=nums[current]<nums[current+1]
count=2
current+=1

#处理后续元素，用is_bigger来记录摆动情况
for i in range(current+1,n):
    if is_bigger and nums[current]>nums[i]:
        is_bigger=False
        count+=1
        current=i
    elif not is_bigger and nums[current]<nums[i]:
        is_bigger=True
        count+=1
        current=i
    else:
        current=i

print(count)
```
