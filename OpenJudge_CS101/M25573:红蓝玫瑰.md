# M25573:红蓝玫瑰
**算法核心：动态规划（二元状态）**  
1）找到对立的两种状态  
2）找两种状态互相影响的递推  
3）由于只与前一项相关，可以不储存数据（空间优化）
```python
roses=input()
n=len(roses)

#设置状态函数
R=[0]*n
B=[0]*n
if roses[0]=='R':
    R[0],B[0]=0,1
else:
    R[0],B[0]=1,0

#进行状态转移
for i in range(1,n):
    if roses[i]=='R':
        R[i]=R[i-1]
        B[i]=min(R[i-1],B[i-1])+1
    else:
        R[i]=min(R[i-1],B[i-1])+1
        B[i]=B[i-1]

print(R[-1])
```
