# E03670:计算鞍点
**算法核心：条件语句**
```python
matrix=[]
for _ in range(5):
    matrix.append(list(map(int,input().split())))

existed=False
for i in range(5):
    for j in range(5):
        if matrix[i][j]==max(matrix[i]) and matrix[i][j]==min(matrix[k][j] for k in range(5)):
            print(f'{i+1} {j+1} {matrix[i][j]}')
            existed=True
if not existed:
    print('not found')
```
