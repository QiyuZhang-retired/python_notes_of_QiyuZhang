# E06374:文字排版
**核心算法：字符串**
```python
n=int(input())
paragraph=input()
words=paragraph.split()

sentence=''
for word in words:
    if len(sentence+word)<=80:
        sentence+=word+' '
    else:
        print(sentence)
        sentence=word+' '

print(sentence)
```
