# E18188:图像的均值滤波
**算法核心：矩阵**
```python
M,N=map(int,input().split())
matrix=[]
for _ in range(M):
    matrix.append(list(map(int,input().split())))

output=[[0]*N for _ in range(M)]
for i in range(M):
    for j in range(N):
        cnt=0
        for di,dj in ((-1,-1),(-1,0),(-1,1),(0,-1),(0,0),(0,1),(1,-1),(1,0),(1,1)):
            if 0<=i+di<=M-1 and 0<=j+dj<=N-1:
                output[i][j]+=matrix[i+di][j+dj]
                cnt+=1
        output[i][j]//=cnt

for row in output:
    print(' '.join(list(map(str,row))))
```
