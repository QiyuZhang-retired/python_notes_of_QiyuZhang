# E23566:决战双十一
**核心算法：条件语句**
```python
n,m=map(int,input().split())

output=0

s={}
for _ in range(n):
    i,si=map(int,input().split())
    if i in s:
        s[i]+=si
    else:
        s[i]=si
    output+=si

output-=(output//200)*30

for j in range(1,m+1):
    qj,xj=map(int,input().split('-'))
    if s[j]>=qj:
        output-=xj

print(output)
```
