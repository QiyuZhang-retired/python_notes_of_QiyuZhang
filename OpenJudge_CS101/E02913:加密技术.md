# E02913:加密技术
**核心算法：ACSII表**
```python
keys=[4,9,6,2,8,7,3]
ASCII_min=32
mod=91

input_str=input()

#进行加密
output_str=[]
for i,c in enumerate(input_str):
    key=keys[i%len(keys)]
    new_c=chr(ASCII_min+(ord(c)+key-ASCII_min)%mod)
    output_str.append(new_c)

print(''.join(output_str))
print(input_str)
```
