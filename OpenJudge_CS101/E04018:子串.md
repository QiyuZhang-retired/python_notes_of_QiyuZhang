# E04018:子串
**算法核心：深度优先搜索**  
也可以用双指针标记，核心一致
```python
def dfs(s,t,i,start):
    if i==len(s):
        return True

    for k in range(start,len(t)):
        if t[k]==s[i]:
            return dfs(s,t,i+1,k+1)

    return False

while True:
    try:
        s,t=input().split()
        if dfs(s,t,0,0):
            print('Yes')
        else:
            print('No')
    except EOFError:
        break
```
