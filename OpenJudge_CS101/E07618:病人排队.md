# E07618:病人排队
**算法核心：排序**
```python
n=int(input())
patients=[]
for i in range(n):
    ID,age=input().split()
    patients.append((ID,int(age),i))

#使用lamda构造排序要求
sorted_patients=sorted(patients,key=lambda p:(0,-p[1],p[2]) if p[1]>=60 else (1,p[2]))

for p in sorted_patients:
    print(p[0])
```
