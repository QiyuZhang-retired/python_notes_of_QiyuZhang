# E02899:矩阵交换行
**算法核心：矩阵**  
注意输出格式：每个数字占四个字符
```python
matrix=[]
for _ in range(5):
    matrix.append(list(map(int,input().split())))
n,m=map(int,input().split())

if 0<=n<=4 and 0<=m<=4:
    row_n=matrix[n][:]
    matrix[n]=matrix[m][:]
    matrix[m]=row_n

    #按格式输出
    for row in matrix:
        for num in row:
            print(f"{num:4d}", end="")
        print()

else:
    print('error')
```
