# T12559:最大最小整数
**核心算法：冒泡排序**
```python
n=int(input())
nums=input().split()

#冒泡排序
for i in range(n-1):
    for j in range(n-i-1):
        if nums[j]+nums[j+1]<nums[j+1]+nums[j]:
            nums[j],nums[j+1]=nums[j+1],nums[j]

print(int(''.join(nums)),int(''.join(list(reversed(nums)))))
```
