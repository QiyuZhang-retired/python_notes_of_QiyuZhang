# M19944:这一天星期几
**算法核心：数学**
```python
weekdays=['Sunday','Monday','Tuesday','Wednesday','Thursday','Friday','Saturday']

n=int(input())

for _ in range(n):
    date=input()
    year=int(date[:4])
    m=int(date[4:6])
    d=int(date[6:])

    if m==1 or m==2:
        m+=12
        year-=1
    c=year//100
    y=year%100
    w=(y+y//4+c//4-c*2+(13*(m+1))//5+d-1)%7

    print(weekdays[w])
```
