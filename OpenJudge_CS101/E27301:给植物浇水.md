# E27301:给植物浇水
**核心算法：条件语句**
```python
n,a,b=map(int,input().split())
plants=list(map(int,input().split()))
a0,b0=a,b

i,j=0,n-1
cnt=0
while i<j:
    if a>=plants[i]:
        a-=plants[i]
    else:
        a=a0-plants[i]
        cnt+=1
    if b>=plants[j]:
        b-=plants[j]
    else:
        b=b0-plants[j]
        cnt+=1

    i+=1
    j-=1

if i==j:
    if a<plants[i] and b<plants[j]:
        cnt+=1

print(cnt)
```
