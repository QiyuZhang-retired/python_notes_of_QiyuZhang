# E23564:数论
**算法核心：条件语句**
```python
n=int(input())

if n==1:
    print(1)
else:
    summary=0
    p=2
    current=n

    while p**2<=n:
        if current%p==0:
            if current%(p**2)==0:
                print(0)
                exit()
            current//=p
            summary+=1
        p+=1
    if current>1:
        summary+=1

    print(1 if summary%2==0 else -1)
```
