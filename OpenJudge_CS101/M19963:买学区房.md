# M19963:买学区房
**算法核心：条件语句**
```python
n=int(input())
places=list(map(eval,input().split()))
distances=[l[0]+l[1] for l in places]
prices=list(map(int,input().split()))
values=[distances[i]/prices[i] for i in range(n)]

#计算两个中位数
sorted_prices=sorted(prices)
sorted_values=sorted(values)
median_prices=0
median_values=0
if n%2==0:
    median_prices=(sorted_prices[n//2-1]+sorted_prices[n//2])/2
    median_values=(sorted_values[n//2-1]+sorted_values[n//2])/2
else:
    median_prices=sorted_prices[n//2]
    median_values=sorted_values[n//2]

#判断中位数
count=0
for j in range(n):
    if prices[j]<median_prices and values[j]>median_values:
        count+=1
print(count)
```
