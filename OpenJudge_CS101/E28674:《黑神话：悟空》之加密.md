# E28674:《黑神话：悟空》之加密
**算法核心：ACSII表**
```python
k=int(input())%26
s=input()

result=''
for c in s:
    if c.islower():
        result+=chr(ord(c)-k if ord(c)-k>=97 else ord(c)-k+26)
    else:
        result+=chr(ord(c)-k if ord(c)-k>=65 else ord(c)-k+26)

print(result)
```
