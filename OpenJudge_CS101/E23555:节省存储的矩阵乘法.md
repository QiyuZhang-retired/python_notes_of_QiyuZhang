# E23555:节省存储的矩阵乘法
**算法核心：矩阵**
```python
n,m1,m2=map(int,input().split())

X=[[0]*n for _ in range(n)]
Y=[[0]*n for _ in range(n)]
for _ in range(m1):
    h,l,x=map(int,input().split())
    X[h][l]=x
for _ in range(m2):
    h,l,y=map(int,input().split())
    Y[h][l]=y

Z=[[0]*n for _ in range(n)]
for i in range(n):
    for j in range(n):
        for k in range(n):
            Z[i][j]+=X[i][k]*Y[k][j]
        if Z[i][j]!=0:
            print(f'{i} {j} {Z[i][j]}')
```
