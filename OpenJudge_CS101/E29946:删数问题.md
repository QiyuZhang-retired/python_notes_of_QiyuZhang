# E29946:删数问题
**算法核心：贪心算法，单调栈**  
由于应该是前面的数字越小越好，那就维护一个单调递增栈直到不能移除数字
```python
n=input()
k=int(input())

#维护单调递增栈
stack=[]
for num in n:
    while k>0 and stack and stack[-1]>num:
        stack.pop()
        k-=1
    stack.append(num)
    
if k>0:
    stack=stack[:-k]
    
print(int(''.join(stack)))
```
