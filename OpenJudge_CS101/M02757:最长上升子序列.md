# M02757:最长上升子序列
**核心算法：动态规划（LIS）**
```python
N=int(input())
nums=list(map(int,input().split()))

#设置状态转移函数并进行转移
dp=[1]*N
for i in range(1,N):
    for j in range(i):
        if nums[j]<nums[i]:
            dp[i]=max(dp[i],dp[j]+1)

print(max(dp))            
```
