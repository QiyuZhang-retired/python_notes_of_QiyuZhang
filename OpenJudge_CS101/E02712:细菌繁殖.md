# E02712:细菌繁殖
**算法核心：元组**
```python
days_in_month = (31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31)
n = int(input())

for _ in range(n):
    a1, b1, x, a2, b2=map(int, input().split())
    days = b2 - b1

    for i in range(a1, a2):
        days += days_in_month[i - 1]

    print(x * 2 ** days)
```
