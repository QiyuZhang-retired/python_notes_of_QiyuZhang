# E02792:集合加法
**核心算法：双指针**  
1）用双指针减小时间复杂度  
2）利用Counter方法生成哈希表处理重复数字
```python
from collections import Counter

n=int(input())
for _ in range(n):
    s=int(input())
    a=int(input())
    A=list(map(int, input().split()))
    b=int(input())
    B=list(map(int, input().split()))

    #排序便于设置双指针，生成哈希表处理重复数字
    A.sort()
    B.sort()
    count_A = Counter(A)
    count_B = Counter(B)
    i, j = 0, b-1
    count = 0

    #进行指针的移动
    while i < a and j > 0:
        if A[i] + B[j] == s:
            count += count_A[A[i]] * count_B[B[j]]
            i += count_A[A[i]]
            j -= count_B[B[j]]
        elif A[i] + B[j] < s:
            i += count_A[A[i]]
        elif A[i] + B[j] > s:
            j -= count_B[B[j]]

    print(count)
```
