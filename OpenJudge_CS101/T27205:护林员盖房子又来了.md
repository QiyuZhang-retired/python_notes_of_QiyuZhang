# T27205:护林员盖房子又来了
**核心算法：单调栈**  
1）压缩矩阵为高度heights  
2）用单调栈，若后续高度低意味着前面的高度到达了右端点，可以弹出并计算面积
```python
m,n=map(int,input().split())
forest=[]
for _ in range(m):
    forest.append(list(map(int,input().split())))

max_area=0
#多开一位记录0来保证后续所有元素出栈
heights=[0]*(n+1)
for i in range(m):
    stack=[]
    for j in range(n):
        #更新高度
        if forest[i][j]==0:
            heights[j]+=1
        else:
            heights[j]=0

    #维护单调递增栈并输出面积
    for j in range(n+1):
        while stack and heights[j]<heights[stack[-1]]:
            index=stack.pop()
            height=heights[index]
            width=j-stack[-1]-1 if stack else j
            max_area=max(max_area,height*width)
        stack.append(j)

print(max_area)
```
