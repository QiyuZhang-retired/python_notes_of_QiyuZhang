# E28681:奖学金
**核心算法：排序**
```python
n=int(input())

students=[]
for i in range(1,n+1):
    a,b,c=map(int,input().split())
    students.append((i,a+b+c,a,b,c))

students.sort(key=lambda x:(x[1],x[2]),reverse=True)

for i in range(5):
    print(f'{students[i][0]} {students[i][1]}')
```
