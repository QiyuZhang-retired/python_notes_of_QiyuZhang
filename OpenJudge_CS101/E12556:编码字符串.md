# E12556:编码字符串
**算法核心：字符串**
```python
word=input().lower()

result=''
current=word[0]
count=0
for c in word:
    if current==c:
        count+=1
    else:
        result+=f'({current},{count})'
        current=c
        count=1
result+=f'({current},{count})'

print(result)
```
