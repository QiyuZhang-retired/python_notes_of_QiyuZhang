# M02766:最大子矩阵
**核心思路：动态规划（Kadane算法）**  
1）降维压缩矩阵  
2）Kadane算法计算LICC问题
```python
N=int(input())

#处理凌乱输入
nums=[]
while len(nums)<N*N:
    nums.extend(input().split())
matrix=[list(map(int,nums[i*N:(i+1)*N])) for i in range(N)]

#压缩矩阵，遍历i到j行列和
max_sum=float('-inf')
for i in range(N):
    col_sum=[0]*N
    for j in range(i,N):
        for k in range(N):
            col_sum[k]+=matrix[j][k]

        #Kadane算法
        dp=col_sum[0]
        max_sum_current=dp
        for x in range(1,N):
            dp=max(dp+col_sum[x],col_sum[x])
            max_sum_current=max(dp,max_sum_current)

        max_sum=max(max_sum_current,max_sum)

print(max_sum)
```
