# E19943:图的拉普拉斯矩阵
**核心算法：矩阵**
```python
n,m=map(int,input().split())
L=[[0]*n for _ in range(n)]

for _ in range(m):
    i,j=map(int,input().split())
    L[i][i]+=1
    L[j][j]+=1
    L[i][j]-=1
    L[j][i]-=1

for row in L:
    print(' '.join(list(map(str,row))))
```
