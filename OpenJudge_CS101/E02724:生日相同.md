# E02724:生日相同
**算法核心：排序**  
1）lambda用法方便排序key的定义  
2）字符串排序和数字排序有不同结果
```python
n = int(input())
birthdays=[]
for _ in range(n):
    birthdays.append(input().split())

#对birthdays合理排序方便输出
birthdays.sort(key = lambda x: (int(x[1]), int(x[2])))

same_birthday = birthdays[0][1:3]
IDs = []

for birthday in birthdays:
    if birthday[1:3] == same_birthday:
        IDs.append(birthday[0])
    else:
        if len(IDs) >= 2:
            print(' '.join(same_birthday + IDs))
        same_birthday = birthday[1:3]
        IDs=[birthday[0]]

#补充输出
if len(IDs) >= 2:
    print(' '.join(same_birthday+IDs))
```
