# E03143:验证“歌德巴赫猜想”
**核心算法：质数**  
数据较小，直接用试除法
```python
#构建判断质数的函数
import math
def is_prime(n):
    if n<=1:
        return False
    if n==2:
        return True
    if n%2==0:
        return False
    for i in range(3,int(math.sqrt(n))+1,2):
        if n%i==0:
            return False
    return True

x=int(input())
if x>=6 and x%2==0:
    for i in range(3,x//2+1,2):
        if is_prime(i) and is_prime(x-i):
            print(f'{x}={i}+{x-i}')
else:
    print('Error!')
```
