# E07743:计算矩阵边缘元素之和
**算法核心：矩阵**
```python
m,n=map(int,input().split())
matrix=[list(map(int,input().split())) for _ in range(m)]

result=0
#计算首尾行
for num in matrix[0]+(matrix[-1] if m > 1 else []):
    result+=num
#计算中间行
for k in range(1,m-1):
    result+=matrix[k][0]+(matrix[k][-1] if n>1 else 0)

print(result)
```
