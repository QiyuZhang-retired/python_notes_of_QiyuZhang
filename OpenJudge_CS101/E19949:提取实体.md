# E19949:提取实体
**算法核心：字符串**
```python
N=int(input())

result=0
for _ in range(N):
    word=input()

    for i in range(len(word)-2):
        if word[i:i+3]=='###':
            result+=0.5
    for i in range(len(word)-6):
        if word[i:i+7]=='### ###':
            result-=1

print(int(result))
```
