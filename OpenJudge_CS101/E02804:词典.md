# E02804:词典
**算法核心：字典**
```python
dictionary={}

#输入字典
while True:
    words=input().split()
    if not words:
        break
    dictionary[words[1]]=words[0]

#按输入提取字典值
while True:
    try:
        word=input()
        if word in dictionary:
            print(dictionary[word])
        else:
            print('eh')
    except EOFError:
        break
```
