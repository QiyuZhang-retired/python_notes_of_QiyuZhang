# 最大连续子序列和
**算法核心：动态规划（Kadane算法）**  
1）状态转移函数：以i个数结尾的子序列的和的最大值  
2）状态转移关系：是使用上一个数字和只使用这个数字的两种情况的较大值
```python
n=int(input())
a=list(map(int,input().split()))

#设置状态函数
dp=[0]*n
dp[0]=a[0]

#进行状态转移
for i in range(1,n):
    dp[i]=max(a[i],dp[i-1]+a[i])

print(max(dp))
```
