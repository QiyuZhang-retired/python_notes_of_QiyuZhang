# 198.打家劫舍
**算法核心：动态规划**   
1）状态转移函数：dp表示偷前i家的最大收益  
2）状态转移关系：应是前i-2家最大值加上此处收益与前i-1家最大值中的较大值
```python
class Solution:
    def rob(self, nums: List[int]) -> int:

        n=len(nums)
        if n==1:
            return nums[0]

        #设置状态函数
        dp=[0]*(n+1)
        dp[1]=nums[0]

        #进行状态转移
        for i in range(2,n+1):
            dp[i]=max(nums[i-1]+dp[i-2],dp[i-1])

        return dp[-1]
```
