# M02945:拦截导弹
**算法核心：动态规划（LIS）**  
1）状态转移函数：以第i个数结尾的最长不递增子列数目  
2）状态转移关系：dp第i值是前i-1个数（高度更高的）中最大的加一
```python
k = int(input())
heights = list(map(int, input().split()))

#设置状态函数
dp = [1] * k

#进行状态转移
for i in range(k):
    for j in range(i):
        if heights[j] >= heights[i]:
            dp[i] = max(dp[j] + 1, dp[i])

print(max(dp))
```
