# M02760:数字三角形
**核心算法：动态规划**  
1）状态转移函数：dp表示在此处结尾的路径的最大数字和  
2）状态转移关系：上一行相邻dp较大的加上此处的数为此处的dp
```python
#输入
N=int(input())
triangle=[]
for _ in range(N):
    triangle.append(list(map(int,input().split())))
#设置dp数组
dp=[[0]*i for i in range(1,N+1)]
#根据状态转移函数填充表格
dp[0][0]=triangle[0][0]
for i in range(1,N):
    dp[i][0]=dp[i-1][0]+triangle[i][0]
    for j in range(1,i):
        dp[i][j]=max(dp[i-1][j-1],dp[i-1][j])+triangle[i][j]
    dp[i][i]=dp[i-1][i-1]+triangle[i][i]
#输出
print(max(dp[N-1]))
```
